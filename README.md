## 🐦 Twitter Clone

A responsive Twitter (X) UI clone built using modern frontend technologies. This project replicates the look and feel of Twitter’s interface, focusing on clean UI design and responsiveness.

---

# 🚀 Live Demo

https://tewitter-clone.vercel.app/

---

## 📌 Features

🐦 Modern Twitter-inspired UI
📱 Fully responsive design (mobile, tablet, desktop)
🧭 Sidebar navigation layout
📰 Tweet feed interface
⚡ Fast and lightweight frontend

---

## 🛠️ Tech Stack

React
Vite
Tailwind CSS
JavaScript
